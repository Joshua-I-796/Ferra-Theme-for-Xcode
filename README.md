# Ferra-Theme-for-Xcode
An Xcode Port for the theme Ferra (huge shout out to the original creator of this theme @casperstorm, the original Ferra theme can be found at https://github.com/casperstorm/ferra/ )

# To Use

Just download and copy the file into the FontAndColorThemes folder in the path:-  ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
